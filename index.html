Prework Study Guide

HTML

  • HTML stands for Hypertext Markup Language.
  • HTML is used to give the content structure.
  • Think of HTML as the sturdy frame we have to build for our code.
  • The head element contains information about the webpage.
  • The li element creates bullet points.
  • The body element represents the visible content shown to the user.

CSS

  • CSS stands for Cascading Style Sheets.
  • CSS is used to give the content style.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Git is used in Gitbash and helps connect our code to Github.
  • The p element represents a paragraph block of text.
  • The ul element stands for unordered lists.
  • The li element stands for list items.
  • The img element contains info about images displayed on the webpage.

JAVASCRIPT

  • JS stands for JavaScript.
  • Java is the most common language used in computer programming.
